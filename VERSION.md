## 版本特性总览

### 核心功能

1. **可变角色数量** - 支持 2-8 个角色，自由调整 ⭐新增
2. **顺序发言机制** - 角色依次发言，对话更真实连贯 ⭐核心特性
3. **交互式群聊** - 用户可以实时参与角色对话
4. **私聊模式** - 与任意角色进行1对1私密对话
5. **实时状态提示** - 显示角色思考状态和生成进度 ⭐新增
6. **双模式支持** - Mock 数据模式 + Gemini API 模式
7. **对话导出** - 导出群聊和私聊记录

### 技术栈

- **前端框架**: Streamlit 1.28.0+
- **AI 模型**: Google Gemini 2.0 Flash
- **Python 版本**: 3.7+

### 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```
